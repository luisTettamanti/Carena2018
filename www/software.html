Oferta Académica

Tecnicatura Superior en Desarrollo de Software

El Técnico Superior en Desarrollo de Software estará capacitado para:

  • Modelar y construir artefactos de software a partir de especificaciones.
  • Verificar la calidad de artefactos de software, diseñando casos de prueba unitarios y de integración.
  • Optimizar y depurar código de artefactos de software para resolver problemas y mejorarlos.
  • Documentar en base a normas y estándares establecidos.
